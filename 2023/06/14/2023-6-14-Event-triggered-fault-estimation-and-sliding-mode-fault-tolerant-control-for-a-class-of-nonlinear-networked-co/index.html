Event-triggered fault estimation and sliding mode fault-tolerant control for a class of nonlinear networked control systems

一类非线性网络控制系统的事件触发故障估计与滑模容错控制

摘要

本文研究了一类具有执行器故障和扰动的离散时间Lipschtiz非线性网络控制系统的事件触发故障估计和滑模容错控制问题。

  1. 首先，设计一个事件触发的故障/状态观测器，用于实时估计系统状态和执行器故障。

  2. 再在状态估计空间中构造离散时间滑动曲面：

    利用重新表述的Lipschitz性质和延迟系统分析方法，考虑事件触发方案、执行器故障、外部干扰和网络诱导延迟，将滑模动力学和状态/故障误差动力学转换为统一的线性变参数(LPV)网络系统模型。在此模型的基础上，借助于Lyapunov-Krasovskii泛函方法，导出保证具有规定H∞性能的闭环系统的稳定性的时滞相关充分条件。

  3. 综合一种基于观测的滑模FTC(容错控制)定律，保障滑动表面的可达性

  4. 仿真分析

引言

背景：传输延迟、有线通信资源、丢包，要开发一种既能降低降低通信负载且保证闭环系统稳定性和性能的传输方案。

— ETC（event-triggered control）方案：在保证系统稳定性和预期性能的同时，节省有限的通信和计算资源

传统时间触发策略：数据包在固定的采样周期内传输

ETC策略的优势在于，只有当违反(或满足，取决于规定的阈值)事件触发条件时，数据包才会在网络上进行传输

FTC可分为两种主要方法：被动与主动

  • 被动FTC (PFTC)方法：将故障视为不确定性的附加形式，具有固定结构的控制器，但当发生的系统故障超出规定的故障集时，其性能不能始终得到保证
  • 主动故障检测(AFTC)方法：主要基于故障检测和隔离(FDI)或估计来补偿已发生的故障，从而保证系统的预期性能

​ FDI可以指示系统故障的发生并确定故障类型，但不易获得准确的故障类型、幅值等信息

故障估计(FE)技术 —估计故障的类型与幅值。在这种方法中，利用估计的故障信号，可以构造一个容错控制器来补偿故障的影响

滑模控制：对不确定性、非线性和外部干扰的鲁棒性

本文针对时变时滞、附加执行器故障和外部干扰的离散Lipschitz非线性网络控制系统，提出了一种事件触发故障估计与滑模滑模容错控制的集成设计框架。

本文主要研究内容：

  1. 采用事件触发方案节省有限的网络资源；设计事件触发状态/故障观测器，在受网络延迟影响的情况下同时估计系统状态和执行器故障
  2. 借助于重新表述的Lipschitz性质和延迟系统分析方法，将滑模动力学和状态/故障误差动力学转化为统一的LPV网络系统模型
  3. 基于Lyapunov-Krasovskii泛函方法，利用线性矩阵不等式(LMI)公式，通过H∞优化得到闭环系统的稳定性和性能分析的充分条件
  4. 考虑事件触发方法，设计了一种基于观测的离散时间滑模FTC律，以保证滑动面的可达性

结论

本文研究了具有执行器故障和外部干扰的离散时间 Lipschitz 非线性NCSs 的事件触发故障估计和滑模容错控制集成设计问题。首先，设计了事件触发状态/故障观测器来同时估计系统状态和执行器故障，然后便于在状态估计空间中设计离散时间滑动面函数。通过重新定义的 Lipschitz 性质和延迟系统分析方法，将闭环系统转化为包含网络诱导延迟、事件触发方案和 SMC 方法影响的LPV 网络控制系统。然后，基于Lyapunov-Krasovskii 泛函方法，导出了保证LPV 系统稳定并具有规定H∞ 性能的时滞相关稳定性条件。在此基础上，合成了一种基于观测的滑模容错控制器，以保证滑模的可达性。最后，通过仿真实验验证了所提方法的有效性。在未来，我们可以将该方法推广到离散非线性网络控制系统的输出跟踪控制问题。
