2.基础知识

常量

作用：记录程序中不可更改的数据

两种定义方式:

  1. #define 常量名 常量值
  2. const 数据类型 常量名 = 常量值；

关键字

其中，sizeof 作用为统计数据类型所占存大小

语法格式：sizeof(数据类型/变量名)

实型（浮点型）

1.单精度float

2.双精度double

默认情况下，输出一位小数会显示出6位有效数字

科学计数法：

3e2：3*10^2

3e-2：30.1*^2

字符型

作用: 显示单个字符

语法: char ch = ‘a‘; (注意为单引号，引号内部只有一个字符而非字符串)

  • 字符型变量只占用一个字节，将字符对应的ASCII码值放入存储单元而非放入字符本身
  • 直接查看字符型变量对应的ASCII码值: cout<<(int)ch<<endl; a–97; A–65

转义字符

\t与其前字符共占8个字符，保证其后左对齐

字符串型

  • char 字符串名[] = “ “ 注意要加 “[]“，等号后用””包含字符串
  • string 字符串名 = “ “ 注意添加头文件 #include

布尔类型

bool类型只有两个值：

  • true 本质为1
  • false 本质为0

所占内存空间为1

eg. bool flag = true

3.程序流程结构

  • 顺序结构 ：程序按顺序执行，不发生跳转
  • 选择结构 ：依据条件是否满足，有选择地执行相应功能
  • 循环结构 ：依据条件是否满足，循环多次执行某段代码

选择结构

if 语句

  • 单行if语句
  • 多行if语句
  • 多条件语句
  • 嵌套if语句

三目运算符

eg:

(a > b ? a : b)

意义：若a>b，返回a，否则返回b

在C++中三目运算符返回的是变量，可以继续赋值

c = (a > b ? a : b)

将a、b中的最大值赋值给c

switch语句

switch结构清晰，但在判断的时候只能为整型，不能是区间

while循环
