Observer-Based Fault Tolerant Control for a Class of Nonlinear Systems via Filter and Neural Network

发表于2023-07-09|更新于2023-07-09
|字数总计:6.1k|阅读时长:18分钟|阅读量:

一类基于观测器的非线性系统的滤波和神经网络容错控制

摘要

针对一类以严格反馈形式表示的非线性系统，在未知系统动力学和执行器故障情况下，提出了一种基于滤波器和神经网络的容错控制策略。首先利用自适应神经网络(ANN)进行状态观测器的设计，从而获得不可测量的系统状态。注意ANN仅在设计状态观测器时使用，而不在设计控制器时使用。在该方法中，引入滤波技术来构造虚拟控制输入，不仅可以减少ANN逼近误差和状态估计误差带来的不利影响，而且还可以解决微分项的展开问题。此外，将反推技术与所提出的NN相结合，设计了容错跟踪控制器，并采用了一种不同于上述ANN的新的权值更新规律。理论分析和仿真结果表明，在存在执行器故障和系统动力学未知的情况下，所提出的FTC策略能够保证跟踪误差收敛到一个小的零区域。

介绍

非线性系统的控制设计一直是一个活跃的话题，并获得了相当大的关注。同时，未知系统动力学和执行器故障的存在使得控制设计变得更具挑战性。在过去的几十年里，各种各样的控制策略被报道用来处理非线性系统的控制问题。

一方面，为了很好地完成控制任务，研究有效、及时的响应控制策略具有重要意义。到目前为止，已经取得了大量的成果。举几个例子，在[5]中，对二阶多飞行器系统进行了研究，提出了一种抗干扰滑模控制器，以保证具有未知扰动的闭环系统的收敛性。在[6]中，针对存在外部时变干扰和内部不确定性的飞机，通过利用鲁棒反馈控制策略提出了一种抗扰跟踪控制方案，使被控系统对干扰和不确定性具有鲁棒性。

在[7]中，通过设计非线性速度观测器和模糊自适应观测器相结合的控制方法，处理了具有未知干扰的垂直起降飞机的跟踪控制问题。通过稳定性分析和仿真结果验证了该方法的有效性。[8]研究了扰动与输出约束不匹配的翼身混合飞机，为了处理扰动，设计了固定时间非递归观测器。然后利用势垒李雅普诺夫函数和非奇异快速终端滑模技术设计控制器。其他类似结果可参考文献[6]、[9]-[12]。

另一方面，确保设备的安全可靠性的研究是极其重要的。随着时间的推移，由于器件的老化，装置不可避免地会发生一些故障。因此，能够适应故障(即组件故障、执行器故障和传感器故障)引起的系统模型变化的容错控制得到了广泛的研究和发展。FTC方法一般分为被动FTC和主动FTC。被动FTC的主要思想是在不改变控制器结构的情况下设计鲁棒控制策略，使被控系统对故障不敏感。而主动FTC可以根据故障调整控制器的参数，甚至改变控制器的结构。由于主动FTC在故障出现时能够更加灵活地保证闭环系统的控制性能，因此受到了越来越多的关注。文献[13]探讨了具有未知执行器故障和不确定性的过驱动飞机的FTC问题，研究了一种新的基于估计量的滑模控制方案，使被控系统在存在执行器故障和不确定性时保持稳定。[14]提出了一种主动FTC策略来处理航天器姿态控制系统发生的执行器故障，该方法可以及时准确地估计和补偿故障。在[15]中，设计了观测器来估计VTOL飞行器发生的软执行器故障。此外，通过实验验证了所提出的FTC方法的有效性。其他容错控制策略可参考[16]-[20]。

可以发现，设计一个能够及时、准确地获取故障信息的观测器，对于保证令人满意的容错性能非常重要。由于神经网络具有逼近连续函数的优良能力，是处理故障的有效工具。[21]中提出了一种将鲁棒技术与人工神经网络相结合的飞机控制方法，该方法可以保证输出跟踪误差收敛到零的小区域。文献[22]针对存在参数不确定性和多干扰的飞机，提出了一种基于自适应神经网络的控制策略。该方法利用自适应神经网络来逼近系统中出现的未知函数，并引入动态面控制来处理退步技术中微分项的展开问题。文献[23]对旋翼无人机进行了研究，引入神经网络来应对外部干扰和测量误差，使得所设计的基于神经网络的控制器对风扰动具有鲁棒性。可以得出，这些方法的一个相似之处是使用李雅普诺夫函数来设计自适应神经网络的权值更新规律。然而，这种方法只能保证神经网络权值的收敛性。因此，如何处理神经网络逼近误差，并进一步提高控制性能是值得研究的问题。

在本工作中，针对一类具有未知动力学、执行器故障和系统状态不可测的非线性系统，提出了一种基于滤波和神经网络的FTC策略。首先，利用人工神经网络来近似系统未知函数，并设计状态观测器来估计不可测系统状态。然后，利用反演技术、滤波技术和具有新权值更新律的神经网络设计了容错控制器。本研究的主要贡献如下：

1） 与文献[24]-[26]相比，本方法在设计状态观测器时仅使用神经网络技术来近似未知函数，而没有将其用于设计控制器。此外，虚拟控制输入通过采用滤波技术设计，其优点是

a）. 控制器设计过程可以简化;

b）. ANN近似误差和补偿状态估计误差，从而可以进一步减小跟踪误差;

c）. 成功克服重复微分引起的微分爆炸问题。

2） 与[27]-[30]中使用的ANN相比，开发了一种具有新颖权值更新规律的NN。该方法通过最小化代价函数将NN权值更新到其相应的满意值，可以更准确地逼近和补偿未知函数和执行器故障。从而进一步提高控制性能。

本文组织如下。第二节给出了问题的陈述和一些初步的说明。第三节给出控制设计和稳定性分析。仿真结果和结论分别在第四节和第五节给出。

结论

本文利用滤波技术和在线学习算法，针对漂移函数未知且执行器故障的非线性系统，提出了一种新的基于神经网络的容错控制策略。所设计的滤波器和神经网络能够成功补偿状态观测器误差、自适应神经网络近似误差和执行器故障。此外，该方法还可以处理由反演技术的导数项引起的扩展问题。理论分析和仿真结果表明，该方法可以有效地处理系统状态不可测的被控系统中出现的时变执行器故障和未知系统动力学问题。虽然所提出的控制方法被证明是有效的，但它不能用于解决控制方向未知的系统的控制问题。
